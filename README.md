# Student Grade Tracker 🎓

A desktop application built with Java Swing to manage student records and grades.

## ✨ Features
- Add student name and grades
- View all student records
- Calculate Average, Highest, and Lowest scores
- Interactive GUI built with Java Swing

## 🛠️ Tech Stack
- Java
- Java Swing for GUI
- OOP Concepts


## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/mgarg616/Student-Grade-Tracker.git`
2. Compile: `javac StudentGradeTrackerGUI.java`
3. Run: `java StudentGradeTrackerGUI`
